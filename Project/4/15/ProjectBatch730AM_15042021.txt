Q) For repository, do we need to write class in Spring Boot?
A) No. You just provide interface with  T=Model class name
   and ID = Primary Key DataType.
  One class is generated with DB Operations code by Spring Boot.

Q) Do we need to start always writing from repository
   or we can also start from controller too?
A) Start in any order, but finally all files must exist
   before code.
   But recomanded order -> REPO->SI->SImpl->Controller
